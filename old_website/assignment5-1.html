Assignment 2: GitHub (Due February 11th)

Finish the in-class GitHub assignment described in the slides. Make sure to complete your pull request and comment on at least two other people's pull requests. Also check out the Course Resources page for more information on using Git and GitHub. We will continue to work through GitHub in future weeks so make sure you are comfortable using it. We are also going to (Re)introduce you to JavaScript! We are hoping that this guide will strengthen your understanding of how JavaScript can be used to make complex programs like Snap! It contains critical information to understand how JavaScript is used as a programming language, and what it can be used to accomplish.
